# Testes Automatizados de API

Este projeto tem como objetivo realizar a automação dos testes da API utilizando as bibliotecas **Mocha**, **SuperTest** e **Chai** referente ao módulo 04 "Testando e automatizando Testes de API" da mentoria 2.0 do Júlio de Lima. 

## Tecnologias utilizadas

- [Mocha](https://mochajs.org/) — Framework de testes JavaScript
- [SuperTest](https://www.npmjs.com/) — Biblioteca para realizar requisições HTTP em testes
- [Chai](https://www.chaijs.com/) — Biblioteca de asserções para validações nos testes
